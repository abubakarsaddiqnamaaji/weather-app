# Weather App

A simple and responsive weather application that provides real-time weather updates for any city.

## Features

* Search weather by city name
* Real-time temperature and conditions
* Clean and responsive UI

## Built With

* HTML
* Tailwind CSS
* JavaScript
* Weather API
